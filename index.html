⚔️ Level 1 — The While Engine
WHILE LOOPS: Keep Going Until...
A while loop runs the same code OVER AND OVER as long as a condition is True. The moment the condition becomes False — it STOPS.

🔵 The Blueprint

The structure of a while loop is simple. It checks the condition BEFORE every single run:

while condition_is_true:
    # This runs again and again...
    # ...until condition becomes False
🔁 Think of it like a game loop: "WHILE the player is alive, keep playing." When they die, the loop stops!

🎮 Real Example — Reactor Cooldown

The reactor temperature starts at 100. Every loop, it drops by 15. The loop runs WHILE temp is above 40:

temp = 100

while temp > 40:
    print("🔥 Temp:", temp)
    temp -= 15 # -= means "subtract 15 each time"

print("✅ Reactor safe! Final temp:", temp)
💡 temp -= 15 is shorthand for temp = temp - 15. Every loop, the variable gets smaller!

☠️ Level 2 — The Infinite Trap
INFINITE LOOPS: The Bug Lord's Trap
This is THE most dangerous mistake in while loops. If the condition NEVER becomes False, the loop runs FOREVER — crashing your program!

Look at this broken code. Can you see why it NEVER stops?

x = 1
while x > 0: # x starts at 1 which IS > 0...
    print("Help I'm stuck!")
    # 💀 x NEVER changes! Loop runs forever!
☠️ The fix: always make sure your loop variable CHANGES inside the loop so it can eventually make the condition False!

✅ The Golden Rule

Every while loop needs THREE things to be safe:

1 — START
Set your variable BEFORE the loop begins.
count = 0
2 — CONDITION
Tell Python WHEN to keep looping.
while count < 5:
3 — CHANGE
Update the variable so the loop can END.
    count += 1

💥 Level 3 — The Break Weapon
BREAK: Emergency Stop!
break is like a big red STOP button. It immediately exits the loop — even if the condition is still True!

💥 How break Works

Sometimes you're looping and something happens that means: "Stop everything right now!" That's when you use break.

hp = 100
round_num = 0

while True: # Loop runs "forever"... until break!
    round_num += 1
    hp -= 30
    print(f"Round {round_num} — HP: {hp}")
    if hp <= 0:
        print("💀 Player defeated!")
        break # ← EXITS the loop immediately!
⚠️ while True: is an intentional infinite loop! It runs forever UNTIL a break fires. This is a very common pattern in games and menus!

⏭️ Level 4 — The Skip Maneuver
CONTINUE: Skip This Round!
continue is different from break. Instead of stopping everything, it just skips the rest of the current loop run and jumps straight to the NEXT one.
💥 break
Exits the loop COMPLETELY. No more runs after this.
🚪 → OUT
⏭️ continue
Skips the rest of THIS run only. Next run still happens.
🔄 → NEXT

⏭️ continue in Action

Print all numbers 1–8, but SKIP number 4 and 6:

n = 0
while n < 8:
    n += 1
    if n == 4 or n == 6:
        continue # ← skip print, jump to next run
    print(n)

# Output: 1, 2, 3, 5, 7, 8
# (4 and 6 are skipped!)
🟢 With continue, code BELOW it in the loop is skipped. That's why print(n) never runs for 4 and 6!
